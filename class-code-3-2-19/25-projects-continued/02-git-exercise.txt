1. make a folder called branching-exercise-2-5-19
	mkdir git-exercise-2-5-19
2. go into that folder
	cd git-exercise-2-5-19
3. initialize a git repository
	git init
4. run a command to check how many branches you have in the repo
	git branch
5. run a command to check any remotes you have in the repo
	git remote -v
